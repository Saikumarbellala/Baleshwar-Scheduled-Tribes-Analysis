# Baleshwar Scheduled Tribes Analysis

## Project Overview
Analysis of Scheduled Tribes census data from Baleshwar district 
using MySQL for data analysis and Power BI for visualization.

## Tools Used
- MySQL Workbench
- Power BI Desktop
- Microsoft Excel

## Dataset
- Source: Census data — Baleshwar District Scheduled Tribes
- Total Records: 1071 rows
- Total Columns: 27

## Project Structure
- data/ — cleaned dataset
- sql/ — all SQL queries
- powerbi/ — Power BI dashboard file
- screenshots/ — dashboard screenshots

## SQL Queries Covered
- Basic queries — filtering and selection
- Intermediate — GROUP BY and aggregations
- Advanced — subqueries and window functions
- Views, Stored Procedures, Rankings

## Dashboard Pages
- Page 1 — Population Overview
- Page 2 — Education Analysis
- Page 3 — Tribe Wise Analysis
